## Overview
Gothic Remake Lockpicking Mod is a helpful tool designed to improve the lockpicking experience in Gothic 1 Remake. It assists players with the lockpicking minigame by providing visual guides, timing assistance, and optional automation features. The mod makes the classic and often frustrating lockpicking system more accessible while preserving the original feel of the game. Perfect for players who want to enjoy the story and exploration without getting stuck on difficult locks.
