## 🔍 Troubleshooting Issues

If the tool does not work, check these common items:

* Resolution: Does your screen match the 1920x1080 standard? The tool relies on pixel recognition. Other resolutions cause errors.
* Permissions: Right-click the application icon and select "Run as administrator." This gives the tool permission to read the game window data.
* Antivirus Software: Sometimes security software flags new applications. Create an exception for the folder where you store the tool.
* Game Modes: The tool does not perform well in Fullscreen Exclusive mode. Switch your display settings in the game menu if you experience crashes or freezes.
* Input Latency: If the tool responds slowly, close unnecessary background applications like web browsers or video players. These consume memory that the solver needs to function correctly.
